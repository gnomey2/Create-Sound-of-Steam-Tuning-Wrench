# Create: Sound of Steam: Tuning Wrench
An addon for sound of steam that lets you set redstone link channels automatically

Right-click a pipe with the wrench, and the correct redstone link frequency block is placed underneath.



## Requirements

- Minecraft Forge 1.20.1
- KubeJS
- Create
- Sound of Steam / Pipe Organs (or compatible pipe blocks)



## Installation

1. Make sure KubeJS is installed and working.
2. Download this package.
3. Copy the included `kubejs/` folder into your instance or server root.
4. Launch the game or restart the server.

**Do not put this in the mods folder.**



## Usage

1. Crafting:
2. Right-click any Sound of Steam pipe with the wrench.
4. A Create redstone link is placed two blocks below the pipe, pre-configured to the correct frequency.
